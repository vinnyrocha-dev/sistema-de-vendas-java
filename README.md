# Sistema de Vendas em Java

Projeto desenvolvido para praticar conceitos de Programação Orientada a Objetos (POO) utilizando Java.

## Estrutura do projeto

O projeto possui as seguintes classes:

- Cliente
- Produto
- Venda
- ItemDaVenda
- Funcionario
- Main

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos

## Objetivo

Este projeto foi criado como exercício prático durante meus estudos de desenvolvimento Back End em Java.
